Q5- Explain CSS box model?

The CSS Box Model is a fundamental concept that describes how every HTML element is structured and how its size and spacing are calculated in the browser.

  • Content - The actual text, image, or content inside the element.
  • Padding - Space between the content and the border.
  • Border - The line that wraps the padding and content.
  • Margin - Space outside the border.

Q6- What is the difference between display none and visibility hidden?

Display none -

  1. Completely removes the element from the page layout.

  2. It does not take up any space on the screen.

  3. It is like the element was never there.

  4. The element is not visible and does not occupy space.

Visibility hidden-

  1. The element is invisible, but it still takes up space.

  2. It behaves like a transparent element.

  3. The element is not visible, but the space it would have used is still there.
