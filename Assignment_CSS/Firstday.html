Q2- Explain inline , internal and external stylesheet?

Inline - Inside an HTML tag using the style attribute.
Internal - inside a style tag within the HTML file's head.
External - In a separate .css file and linked using a link tag in the HTML head.

flex-end - Aligns items to the right (end.)

space-between - Equal space between items only.

Q4- Difference between absolute and relative positioning?

Absolute - Moves from nearest positioned ancestor or page.
Relative - Moves from original place.

Q5- What is the difference between padding and margin?

Padding - It is the space inside an element, between its content and border.
Margin - It is the space outside the element, between the element and others.

Q6- What is the z-index in CSS?

It controls the stacking order of elements on the z-axis (front-to-back).
Higher z-index = appears on top.
Lower z-index = stays behind.
